Creating an Object

To create a new object, use:
var newObject = new GameObject(50, 50)
This will create an object at position (50,50), near the top left of the sceen. These values are in pixels and can be changed to place the object at other places on the screen.
In addtion, the obect can be modified to have a size (in pixels) and to be visible.
newObject.xSize = 32; newObject.ySize = 32; newObject.visible = true;
A renderer must be added to the object to give it a graphical representation. For example:
newRend = new SolidRenderer(newObject, 'black')
This will make the object show up as a black square.

Adding Behavior

To make the above object move you will have to add a component to it.
The only required part of a component is an update function. The below code will create a move script.
var moveScript = {update: move}

function move () {
  this.parent.x += 1
}
"this.parent" will refer to the game object it is attached to. So by using "this.parent.x" the position of the object is accessed. Update is called once per frame resulting in the object moving.
To add the script to the Object, use:
newObject.addScript(moveScript)

More Objects and components can be added to create a game.
