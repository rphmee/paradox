Engine Base
canvas:
Variable.
Loaded automatically as part of initializing the engine.
Used to get information about the canvas. For example: width and height.

worldObject:
Variable.
Base of the tree of ordered Objects

camera:
Variable.
Cameras are used to make moving around the game world easier.
Leaving it as undefined draws directly to the canvas.

GameObject
Class.
constructor (xpos, ypos, rot, anchorX, anchorY, scaleX, scaleY, dirX, dirY, attached)
Represents all objects in the game. Can attach components to a GameObject.

Player
Class.
constructor (id)
Creates a Player object with an id and a default Controller.

Controller
Class.
constructor ()
Creates a Controller object that holds a mapping of input keys to actions. Controllers can vary from player to player.

input:
Variable.
Handles input for the player.

function preloadArtAssets (path, names, callback)
Loads the desired assets from their location in storage.

Renderers
SpriteSheetRenderer
Class.
constructor (gameObject, imageSource, depth, anchorX, anchorY, sprites, xSize, ySize, height, width, orientation, delay)
Handles the rendering of a specific animated sprite sheet.

ImageRenderer
Class.
constructor (gameObject, imageSource, depth, anchorX, anchorY, drawAsUI)
Handles the rendering of a specific image.

SolidRenderer
Class.
constructor (gameObject, color, shape, depth, drawAsUI)
Handles the rendering of a basic shape with a flat color.

TextRenderer
Class.
constructor (gameObject, text, font, size, color, styles, depth, drawAsUI)
Handles the rendering of text.

Component
More of a blueprint than a portion of the code.
An object with the update field set to a function. When it is attached to gameobject it will be called once per frame. The component can access the gameobject using this.parent.
Utility

Grid
Class.
A class for initializing a 2 dimensional array and initializing the content to a given value.

Vector2
Class.
Object with X, Y coordinates with access to commonly used Vector operations.

Physics
Class
constructor (gameObject, dx, dy, target)
A Component for adding basic physics behavior to a object

RectCircleColliding
function RectCircleColliding (circleX, circleY, circleRad, rectX, rectY, rectW, rectH, rot)
Checks for overlap between a circle and a rectangle. Used in collision detection but also useful for other cases.

removeGameObject
function removeGameObject (current, gameObject)
Removes a gameObject and its renderers from the game.

removeChild
function removeChild(parent, child)
Removes a child from its parent, but does not remove the child from the game.

findObjectByTag
function findObjectByTag (current, tag)
Finds the first gameObject with a given tag and returns it.

findAllObjectsByTag
function findAllObjectsByTag (current, tag)
Finds all objects with a given tag and returns them as an array.

getObjectUnderMouse
function getObjectUnderMouse (current)
Finds the first object under the mouse pointer and returns it.

getCollision
function getCollision (xpos, ypos, testObject, current)
Check for overlap collision. Returns the colliding gameObject.

draw_rect_border
function draw_rect_border (xStart, yStart, w, h, lineWidth, col)
Draws a rectangular border.

screenSpaceToWorldSpace
function screenSpaceToWorldSpace (screenSpace)
Function for converting a position on the screen to a position in the world when a camera is being used.

setCookie
function setCookie (cookiename, cookievalue, experationdays)
Sets a cookie to a given value. Negative values of days clear the cookie.

getCookie
function getCookie (cookiename)
Get a previously saved cookie.

angleToVector
function angleToVector (angle)
Converts an angle into a Vector2.

raycast
function raycast (v1, v2)
Gets the difference between the two given vectors creating a ray from v1 to v2.

degrees_to_radians
function degrees_to_radians (degrees)
Convert degrees to radians.

radians_to_degrees
function radians_to_degrees (radians)
Convert radians to degrees.

getWorldPosX
function getWorldPosX (obj)
Get the x component of the object's position in the world.

getWorldPosY
function getWorldPosY (obj)
Get the y component of the object's position in the world.

turnToTarget
function turnToTarget (obj, targX, targY)
Turn the gameObject (obj) to face target X, Y (targX, targY)

rotateComponents
function rotateComponents (obj, rot)
Rotate the anchor point of gameObject's (obj) children and renders/children to follow rotation.
